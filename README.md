# Chemini: Advances in Chemistry with AI 

**Chemini** is an interactive web application designed to mimic Google's Gemini interface. It serves as an immersive, AI-driven presentation exploring how Artificial Intelligence is transforming modern chemistry from molecular design to retrosynthesis.

## 🚀 Project Overview

[cite_start]This project bridges the gap between conventional trial-and-error experimentation and modern data-driven chemical discovery[cite: 3]. [cite_start]Instead of a standard slideshow, **Chemini** functions as a permanent web application where an AI simulation (modeled after the creator) explains complex research topics interactively[cite: 21, 23]. You're about to enter an experience like no other. The most immersive experience. 

[cite_start]When opened, the application begins an auto-running presentation in a "chat-like" format, simulating a live AI terminal[cite: 22, 26]. [cite_start]It features a fully integrated look at **ChemDFM**, a Large Language Model (LLM) specifically trained on chemistry data[cite: 4, 24]. Moreover, the UI is impeccable and made to help users become fascinated by STEM. 

## 🧠 Core Research Questions

Chemini addresses six fundamental questions regarding the evolution of AI in chemistry:

1.  [cite_start]**Definitions:** What is an AI model, and what types are used in chemistry today? [cite: 7]
2.  [cite_start]**Evolution:** How have AI models evolved, and what does "AI" mean in a modern chemical context? [cite: 8]
3.  [cite_start]**Prompting:** What is a "system prompt," and how is it used for chemical reasoning in LLMs? [cite: 9]
4.  [cite_start]**Synthesis:** How do LLM-augmented programs automate and optimize reaction planning? [cite: 10]
5.  [cite_start]**Comparison:** What makes **ChemDFM** a breakthrough compared to earlier protein-folding models like **AlphaFold**? [cite: 11]
6.  [cite_start]**Retrosynthesis:** How can AI enhance retrosynthesis through atom-level reasoning? [cite: 12]

## 📚 Chemistry Curriculum Connections

[cite_start]This project integrates directly with the following Chemistry units[cite: 29, 30]:

* **Equilibrium:** Using LLMs to predict stable reaction states.

## ✨ Features

* [cite_start]**Interactive Interface:** Mimics the Google Gemini UI ("Chemini") for a familiar, high-tech user experience[cite: 21].
* [cite_start]**AI Simulation:** Viewers can type questions to receive explanations from an "AI version" of the researcher[cite: 23].
* [cite_start]**Visualizations:** Embedded code blocks and molecule visualizations to illustrate concepts like protein folding[cite: 27].
* [cite_start]**Real-Time Narration:** Summaries are narrated in real-time text within the chat window[cite: 27].
